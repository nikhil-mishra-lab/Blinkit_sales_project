# Blinkit Sales Data Analysis Project

## Project Overview
This project analyzes Blinkit's sales data to uncover key insights regarding revenue generation, customer satisfaction, item types, and outlet performance.

## Key Insights & Features Tracked
- **Total Revenue & Average Sales**: Understanding the overall financial growth.
- **Item Type Analysis**: Identifying top-selling product categories (Fat content vs Regular).
- **Outlet Performance**: Comparing different outlets based on their establishment year, size, and location type (Tier 1, Tier 2, Tier 3).
- **Interactive Visuals**: Developed using dynamic Pivot Tables and clean data practices.

## Tech Stack Used
- **Microsoft Excel**: Data Cleaning, Pivot Tables, and Dashboard design.
